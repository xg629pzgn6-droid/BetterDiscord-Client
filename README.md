## üíª Usage

After installation, launching BetterDiscord-Client is simple:

1. Open Discord as usual.
2. You will notice some new options available in the settings menu.
3. Explore the themes and plugins sections to customize your Discord experience.

### Common Actions

- **Change Theme**: 
    1. Go to User Settings.
    2. Select "Themes" to browse available options.

- **Add Plugins**:
    1. Go to User Settings.
    2. Select "Plugins" and click "Open Plugins Folder".
    3. Drag and drop your downloaded plugins into this folder.
